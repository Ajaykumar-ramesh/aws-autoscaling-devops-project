DevOps Auto Scaling Project

Infrastructure Provisioned with Terraform | Configuration Managed by Ansible

Project Overview

This project demonstrates a production-style DevOps architecture deployed on AWS. The infrastructure is fully automated using Infrastructure as Code principles.

Technologies Used

AWS Terraform Modules Ansible Auto Scaling Group Application Load Balancer Nginx Dynamic Inventory

Architecture

  • Custom VPC with Public Subnets
  • Application Load Balancer distributes traffic
  • Auto Scaling Group launches EC2 instances
  • Ansible automatically installs and configures Nginx

Deployment Flow

  1. Terraform provisions AWS infrastructure
  2. Auto Scaling launches instances
  3. Ansible configures servers automatically
  4. Traffic served through ALB
